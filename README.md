# ✍️ Markdown 编辑器 | MDWrite

一个优雅、快速、功能齐全的双栏 Markdown 编辑器。在左侧书写，在右侧实时预览渲染后的效果，助你专注于创作本身。

**🌐 在线直达链接：[https://mdwrite.netlify.app/](https://mdwrite.netlify.app/)**

---

## ✨ 核心功能

### 1. 📝 沉浸式编辑与实时预览
- **双栏联动**：左侧为纯文本编辑区，右侧为即时渲染的HTML预览区。
- **实时同步**：每键入一个字符，预览都会即刻更新，写作流畅无打断。
- **专注模式**：支持隐藏预览区，进入全屏纯编辑模式。

### 2. 🛠️ 丰富的格式化工具栏
一键插入或环绕所选文本，支持以下 Markdown 语法：
- **文本样式**：加粗 (`**B**`)、斜体 (`*I*`)、删除线 (`~~S~~`)。
- **标题**：快速插入 H1 到 H3 级标题。
- **列表**：有序列表与无序列表。
- **引用**与 `行内代码`。
- **链接** 与 **图片** 嵌入。
- **代码块**：支持语法高亮（通过Highlight.js实现）。

### 3. 💾 便捷的文件管理
- **本地持久化**：利用浏览器 `localStorage` 自动保存内容，防止意外丢失。
- **新建/清空**：一键创建全新文档。
- **导入文件**：支持从本地上传 `.md` 或 `.txt` 文件进行编辑。
- **导出文件**：
  - 下载为 `.md` 文件。
  - 导出为完整的 `.html` 单文件。
  - 一键复制渲染后的 HTML 代码。
